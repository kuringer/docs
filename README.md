# Venalio Dokumentácia

Mintlify docs pre Venalio e-commerce platformu.

## Stav

Dokumentácia je momentálne **nepublikovaná**. V Mintlify dashboarde je deployment branch prepnutý na `maintenance`, ktorý zobrazuje len placeholder stránku.

## Ako obnoviť docs

1. Otvoriť [Mintlify Dashboard](https://dashboard.mintlify.com)
2. Vybrať Venalio projekt
3. **Settings** → **Git Settings**
4. Zmeniť **Branch** z `maintenance` na `main`
5. Save — Mintlify automaticky redeployne plnú dokumentáciu

## Branches

- `main` — plný obsah dokumentácie (všetky stránky, navigácia, obrázky)
- `maintenance` — placeholder stránka "Dokumentácia je dočasne nedostupná"
