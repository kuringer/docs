## Produkty a katalóg

| Pojem | Popis |
|-------|-------|
| **Produkt** | Tovar ponúkaný na e-shope |
| **Kategória** | Skupina produktov podľa typu |
| **Parameter** | Vlastnosť produktu (farba, veľkosť...) |
| **Hodnota parametra** | Konkrétna hodnota parametra (červená, XL...) |
| **Značka** | Výrobca alebo značka produktu |
| **Štítok** | Vizuálny štítok na produkte (Novinka, Akcia...) |
| **Cenová hladina** | Cenová skupina pre rôzne typy zákazníkov |
| **Skladová varianta** | Varianta produktu s vlastnou skladovou zásobou |
| **Sklad** | Dostupnosť produktu na sklade |

## Objednávky

| Pojem | Popis |
|-------|-------|
| **Objednávka** | Nákup vytvorený zákazníkom |
| **Stav objednávky** | Aktuálny stav spracovania objednávky |
| **Košík** | Dočasný zoznam produktov pred objednávkou |
| **Pokladňa** | Proces dokončenia objednávky |
| **Zákazník** | Registrovaný alebo neregistrovaný kupujúci |
| **Kategória zákazníka** | Skupina zákazníkov s osobitnými podmienkami |

## Doprava a platby

| Pojem | Popis |
|-------|-------|
| **Doprava** | Spôsob doručenia objednávky |
| **Úroveň dopravy** | Cenová úroveň dopravy podľa podmienok |
| **Prepravca** | Externá služba zabezpečujúca doručenie |
| **Zásielka** | Konkrétna zásielka odoslaná prepravcom |
| **Platba** | Spôsob úhrady objednávky |
| **Úroveň platby** | Cenová úroveň platby podľa podmienok |

## Financie

| Pojem | Popis |
|-------|-------|
| **Faktúra** | Daňový doklad k objednávke |
| **Číselný rad faktúr** | Pravidlo generovania čísel faktúr |
| **DPH** | Daň z pridanej hodnoty |
| **Sadzba DPH** | Percentuálna sadzba dane |

## Marketing

| Pojem | Popis |
|-------|-------|
| **Kupón** | Zľavový kód aplikovateľný na objednávku |
| **Zľava** | Automatická alebo manuálna zľava na produkty |
| **Newsletter** | Hromadný email zaslaný odberateľom |
| **Vernostný program** | Bodový systém pre zákazníkov |
| **Odporúčanie** | Program odporúčaní medzi zákazníkmi |

## Obsah a nastavenia

| Pojem | Popis |
|-------|-------|
| **Obsahový prvok** | Stavebný blok stránky (text, banner, galéria...) |
| **Web** | Konkrétna doména/jazyková mutácia e-shopu |
| **Jazyková mutácia** | Jazyková verzia webu |
| **Téma** | Vizuálny vzhľad e-shopu |
| **Médiá** | Obrázky, videá a súbory v systéme |
| **SEO** | Optimalizácia pre vyhľadávače |
| **Presmerovanie** | URL presmerovanie (301/302) |
| **Predajňa** | Fyzická predajňa alebo výdajné miesto |

## Systém

| Pojem | Popis |
|-------|-------|
| **Prístupové práva** | Riadenie prístupu administrátorov |
| **Webhook** | Automatické HTTP notifikácie pri udalostiach |
